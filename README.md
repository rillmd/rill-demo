# Demo Vault

This is a demo knowledge repository for [Rill](https://github.com/rillmd/rill), a voice journal and knowledge management system powered by Claude Code.

It showcases what a Rill vault looks like after a few days of active use by a software engineer working on API design, LLM integration, and remote collaboration.

## What's inside

- **inbox/** -- Raw journal entries, meeting notes, and web clips
- **knowledge/** -- Distilled notes, people, and project entities
- **workspace/** -- Active workspaces with deliverables
- **tasks/** -- Task tickets
- **reports/** -- Daily notes generated by `/briefing`
- **pages/** -- Human-facing aggregated views

## Getting started

Install Rill and point it at this directory, or browse the Markdown files directly to understand the structure.
