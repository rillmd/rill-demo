# reports/newsletter/ — Daily Research Reports

Daily research reports generated by `/newsletter` (ADR-031).

## Filename

`YYYY-MM-DD.md` (with versioning: `YYYY-MM-DD-1.md`, etc.)

## Frontmatter

```yaml
---
created: 2026-04-07T07:00+09:00
type: newsletter
date: 2026-04-07
keyword-count: 8
---
```

## Generation Process

1. Extract **up to 8 keywords** from active workspaces / knowledge / tasks / journals
2. Also reference the Watch section (Competitors + Keywords) in `knowledge/projects/*.md`
3. Fetch top 5 results via WebSearch + WebFetch
4. Generate a prose report (fact → interpretation → implication 3-layer structure, 400+ chars/section)

## 3-Layer Context Model

Based on user feedback:
- **Identity** — User's core interests (from knowledge/me.md)
- **Attention** — Current focus (active workspaces, recent journals)
- **Impulse** — Spontaneous interests (recent searches, etc.)

The report integrates these 3 layers.

## Alert Rules

The Alert section covers **information from the past 2 weeks only**. Older information does not go in Alerts.

## Search Target

reports/newsletter/ is a search target for `/focus`, `/briefing`.

## Generation

```bash
/newsletter
```

Manual creation is not recommended.
