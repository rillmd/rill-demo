# inbox/tweets/ — Tweet Archive

Archived tweets. source-type: tweet.

## Filename

`YYYY-MM-DD-HHmmss-author.md` (e.g., `2026-04-07-101757-dzhng.md`)

Auto-generated by `rill clip-tweet <URL>` or `rill sync-twitter`.

## Raw Frontmatter

```yaml
---
created: 2026-04-07T10:17+09:00
source-type: tweet
url: https://twitter.com/user/status/XXXXXX
author: @username
---

Tweet body text...
```

## Organized Frontmatter

```yaml
---
created: 2026-04-07T10:17+09:00
source-type: tweet
url: https://twitter.com/user/status/XXXXXX
author: @username
original-file: inbox/tweets/2026-04-07-101757-username.md
tags: [ai, tools]
mentions: [orgs/anthropic]
---

Summary of tweet + commentary...
```

## Characteristics

- **`url`**: Link to the original tweet (required)
- **`author`**: @username format
- Tweet body is auto-fetched (via FixTweet API)
- `/distill` handles organizing quotes and references

## Immutability Principle

Original files are read-only.

## Plugins

- `/clip-tweet` — Ingest a single tweet
- `/sync-twitter` — Bulk sync from bookmarks, etc.
