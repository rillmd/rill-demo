# inbox/web-clips/ — Web Clips

Clipped web pages. source-type: web-clip.

## Filename

`YYYY-MM-DD-HHmmss-title-slug.md` (e.g., `2026-04-06-173501-github-paperclip-open-source.md`)

Auto-generated by `rill clip <URL>`.

## Raw Frontmatter

```yaml
---
created: 2026-04-06T17:35+09:00
source-type: web-clip
url: https://example.com/article
title: "Article Title"
---

Article body (converted to Markdown)...
```

## Organized Frontmatter

```yaml
---
created: 2026-04-06T17:35+09:00
source-type: web-clip
url: https://example.com/article
title: "Article Title"
original-file: inbox/web-clips/2026-04-06-173501-article.md
tags: [ai, research]
mentions: [orgs/acme-corp]
---

Formatted summary + key points...
```

## Characteristics

- **`url`** and **`title`** are required
- Body text is extracted via Readability-like processing and converted to Markdown
- Binaries containing PII (images, PDFs) may be excluded via `.gitignore` (ADR-047)

## Immutability Principle

Original files are read-only. Only `_organized/` creation is allowed.

## Plugins

`rill clip <URL>` is the ingestion entry point. Chrome Extension support is planned for the future.
